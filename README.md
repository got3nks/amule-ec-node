# aMule EC Protocol - Node.js Implementation

This project provides a Node.js implementation of the aMule External Connector (EC) Protocol based on the [aMule EC Protocol HOWTO](https://wiki.amule.org/wiki/EC_Protocol_HOWTO).

Tested with aMule v2.3.3.

## Features

- Communicate with aMule/EMule via EC protocol
- Send commands and receive status or transfer info

## Getting Started

### Installation

```bash
git clone https://github.com/got3nks/amule-ec-node.git
cd amule-ec-node
npm install
```

### Example Usage

```bash
npm start
```

See [`examples/test.js`](examples/test.js) for usage.
